# Ghostnote Project Hub

Contextual note-taking layer for macOS and creative provenance for mobile.

## 🚀 Key Integrations
- **File System Annotation**: Notes are tied to specific file paths. Works with Dropbox, iCloud, and Google Drive.
- **Evernote Sync**: Seamless backup and searchability. Access via Preferences > Account.
- **Mobile Handwriting**: Use the Ghostnote — your idea app for handwritten ink notes and item provenance.

## 🛠 Setup & Launch
1. **Desktop**: Install Ghostnote 2, grant Accessibility permissions, and use the menu bar icon to start "ghosting" notes to active apps.
2. **Document Support**: Click the cogwheel in any note and select "Install Document Support" to enable script-based tracking for individual files.
3. **Mobile**: Download the iOS app to scan items and add handwritten "origin stories" to your collection.

## 📝 Final Snapshot
- **macOS**: Floating contextual notes that change based on the active Window/File/URL.
- **iOS**: Location-based and handwritten notes focused on item history and sketches.
