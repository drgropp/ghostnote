# GhostNote

GhostNote is a developer-focused productivity ecosystem built around fast notes, terminal workflows, and project memory.

It is designed for people who want simple tools that help them capture ideas, track work, and stay organized without getting buried in bloated apps.

## The GhostNote Ecosystem

GhostNote is planned as a collection of connected tools:

### GhostNote CLI
A command-line note-taking tool for fast capture and project notes.

### GhostNote TUI
A terminal user interface for browsing, editing, and managing notes without leaving the terminal.

### GhostNote Web
A lightweight web version for viewing and managing notes in the browser.

## Current Status

GhostNote is early but actively being developed.

The goal is to keep the core experience fast, simple, and useful while slowly building a larger ecosystem around it.

## Philosophy

GhostNote should be:

- Fast
- Minimal
- Useful
- Terminal-friendly
- Easy to understand
- Flexible enough to grow

No unnecessary clutter. No giant productivity system. Just tools that help you keep track of what matters.
